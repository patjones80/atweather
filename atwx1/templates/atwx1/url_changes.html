December 2024

Effective January 1, 2025 – there will be minor changes to AT Weather in terms of functionality. Past users of the site know that we had a dropdown box to select either the AT or PCT. Selecting one or the other filtered the other two lists. This dropdown box is now removed. The functionality of the dropdown box for selecting trail/state remains the same; making a selection will continue to filter the main location dropdown list.

Related to this is that the URL structure for the site has changed. The URLs for forecast pages used to have parameters called “myTrail”, “myState”, and “myShelter” which determined how all of the dropdown boxes behaved. The last part of the URL now looks, for example, like this:

/forecast?location_id=c738681b-aaa6-47f1-96f1-6ca85cdce033

This means that if you had previously bookmarked an old URL, it will no longer function. If you wish to bookmark certain pages for the future, please select them from the main location dropdown and re-bookmark them. These changes were made in order for the URL to be simpler, and for the code that comprises AT Weather’s behind the scenes (or “back end”) functionality to be more concise.

Please feel free to ask me about any of these changes. I look forward to providing an updated version of AT Weather for the 2025 thru-hiking season, its eleventh year in service. Thank you.
